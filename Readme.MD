\# MikasTuner 🎸



A guitar tuner application built with MFC/C++ for Windows.



\## Features



\- Real-time pitch detection using autocorrelation

\- Visual tuning bars for each guitar string (E, A, D, G, B, E)

\- Microphone selection

\- Noise filtering and signal smoothing



\## Guitar Strings (Standard Tuning)



| String | Frequency |

|--------|-----------|

| E (low) | 82.41 Hz |

| A | 110.00 Hz |

| D | 146.83 Hz |

| G | 196.00 Hz |

| B | 246.94 Hz |

| E (high) | 329.63 Hz |



\## Building



1\. Open `MikasTuner.sln` in Visual Studio 2022

2\. Build solution (Ctrl+Shift+B)



\## Requirements



\- Windows 10/11

\- Visual Studio 2022 with MFC support

\- Microphone
